## Getting Started
**Ruby 3.2.2**
**Rails 7.0.8.4**  
**gem Kaminari 1.2.2 & rack-cors 2.0.2**
**sqlite 3.42.0**

- Clonar repositorio github
***git@github.com:Alefuentes982/pokeApi_Back_ROR_CC.git***

- Instalar dependencias
***bundle install***

- Configurar base de datos (sqlite)
***rails db:create***
***rails db:migrate***

- Ejecutar app 
***rails server -b 192.168.xxx.xx***
